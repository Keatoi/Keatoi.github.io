Non-AlphaNumerical Bubble Sort
Made for a Christmas break challenge during my Master's year. A Bubble sort algorithm that has no words or numbers other than main and a single define.
Project Details
Original Algorithm below the completed version
This project started out by making a bubble sort algorithm to test if it works (much easier to debug code when it has words in it!!). Then defining a decltype and combining it with the ASCII value of ! to give us an example of an integer. Similarly, we can also use an empty Lambda function ({} ()) to make recursive void functions, replacing the for statements and conditionals to replace any if statements.
The numbers were replaced by adding or subtracting ASCII values from each other, and various lengths of _ were used for variable names.
